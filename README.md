# Mini AI Agent

![Python](https://img.shields.io/badge/python-3.10-blue)
![License](https://img.shields.io/badge/license-MIT-green)

A simple open-source AI agent built with Python that supports basic chat and fundamental math & natural science knowledge.

一个使用 Python 实现的简单开源 AI Agent，支持：

- 简单对话
- 基础数学知识
- 基础物理知识
- 基础化学知识
- 基础生物知识
- 基础天文知识

This project is designed for learning how a simple AI agent works.

---

## Features

- Simple chat interaction
- Basic scientific knowledge base
- Lightweight rule-based AI agent
- Easy to understand and extend
- Beginner friendly project

## How It Works

The AI agent works using a simple rule-based system.

Workflow:

1. User enters a question
2. Agent detects intent
3. Agent selects knowledge domain
4. Agent searches knowledge base
5. Agent returns response

This simple architecture is designed for educational purposes.
